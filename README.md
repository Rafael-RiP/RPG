# Sistema de Ficha para RPG

## 📌 Sobre o Projeto

Este projeto consiste em uma **interface web para fichas de personagens de RPG**, permitindo organizar informações de personagens de forma visual e estruturada.

O sistema foi desenvolvido utilizando tecnologias web básicas e tem como objetivo facilitar o gerenciamento de fichas de personagens em campanhas de RPG.

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

## 📂 Estrutura do Projeto

```
RPG-main/
│
├── index.html        # Página principal
├── ficha.html        # Página de ficha do personagem
├── vampiro.html      # Página relacionada ao sistema Vampiro
├── ordem.html        # Página adicional do sistema
├── style.css         # Estilos da aplicação
├── ficha-vampiro.js  # Script de manipulação da ficha
└── README.md         # Documentação do projeto
```

## ⚙️ Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

2. Abra a pasta do projeto.

3. Execute o arquivo:

```
index.html
```

Basta abrir o arquivo no navegador.

## 🎮 Funcionalidades

* Interface visual para fichas de personagens
* Organização de informações importantes do personagem
* Estrutura voltada para campanhas de RPG
* Manipulação de dados utilizando JavaScript

## 🎯 Objetivo do Projeto

O projeto foi desenvolvido com fins educacionais, com foco em **prática de desenvolvimento web utilizando HTML, CSS e JavaScript**, além de oferecer uma ferramenta simples para organização de personagens em jogos de RPG.
